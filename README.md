 CampusConnect

CampusConnect is a web-based platform that helps students discover upcoming campus events and register online. The system provides an easy way for students to stay informed about activities happening within the university.

 Features

- View upcoming campus events
- Event details and schedules
- Student event registration
- Responsive and user-friendly interface
- Easy navigation between pages

 Technologies Used

- HTML5
- CSS3
- JavaScript
- Git & GitHub

 Project Structure
 campus-website/
│
├── landingPage.html # Home page
├── events.html # Displays available events
